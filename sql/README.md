# 📓 SQL Documentation

This folder contains all SQL queries used throughout the Bellabeat Fitness Data Analysis project

## 🏗 Folder Structure

### 01_data_exploration
- Initial exploration and validation of raw datasets

### 02_data_cleaning
- Data cleaning, standardization, and preparation of analysis-ready tables

### 03_analysis
- Business analyses, behavioral insights, and correlation studies

### 00_exploration_template.sql
- Reusable SQL framework used to explore and validate new datasets

For clarity, only exploration queries that contributed to data validation, cleaning decisions, or subsequent analyses are included in this repository
